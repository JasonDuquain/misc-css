A Pen created at CodePen.io. You can find this one at http://codepen.io/hbuchel/pen/qOxGzW.

 Playing around with the new css grid module spec and flexbox. In Chrome, you'll have to head to chrome://flags and enable "Experimental Web Platform Features" for this layout to work. I'm sure it looks pretty not great in all other browsers.

This article was a lot of help: https://hacks.mozilla.org/2015/09/the-future-of-layout-with-css-grid-layouts/

I had a lot of trouble sizing some of the headings and this has made me wish even more that we could size elements to their parent's width, not just the viewport.
