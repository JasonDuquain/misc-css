In the JS panel, uncomment the findMorphOrigin() helper function and you'll get a GUI that'll let you drag the start and/or end origin to new positions and see how it affects the morph. Feel free to copy the findMorphOrigin() function into your own project, and then simply feed in the start and end shapes like findMorphOrigin( ".startSelector", ".endSelector" );. Then use the resulting origin value for your tween(s).
